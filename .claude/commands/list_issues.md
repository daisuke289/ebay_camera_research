# 概要

現在のリポジトリのオープンなIssue一覧を取得して表示する。

# 前提条件

- GitHub MCPがインストールされ、利用可能な状態であること
- リポジトリ: daisuke289/ebay_camera_research

# 処理フロー

1. **Issue一覧の取得**: GitHub MCPの `list_issues` ツールを利用して、オープンなIssueを取得
2. **一覧の整形表示**: 見やすい形式で一覧を表示

# 出力形式

```
# オープンなIssue一覧

| # | タイトル | ラベル | 作成日 |
|---|---------|--------|--------|
| 1 | 〇〇機能の追加 | enhancement | 2025-01-15 |
| 2 | △△のバグ修正 | bug | 2025-01-10 |
```

# エラーハンドリング

- Issue取得に失敗した場合、エラーメッセージを表示
- Issueが0件の場合、「オープンなIssueはありません」と表示
